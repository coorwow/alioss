#alioss
<br/>为thinkcmf5开发的文件上传到阿里云存储的插件
<br/>参考官方七牛云存储，做了一些细节优化
<br/>同时利用阿里图片处理参数，自动支持头像及各比例的缩略图。
<br/>插件使用用方法参考官方文档 <a href="https://www.kancloud.cn/thinkcmf/faq/493510">插件安装</a>
